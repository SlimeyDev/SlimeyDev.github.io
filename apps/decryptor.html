Decryptor:

A simple program to decrypt string data.

This program was written by SlimeyDev in Python, you can find the source code below.



Source code:


from cryptography.fernet import Fernet
import time

encryptedstr = input("Enter the encrypted byte string: ")
encryptedstr = bytes(encryptedstr, "utf-8")
key = input("enter the key: ")
key = bytes(key, "utf-8")

fernet = Fernet(key)

decstr = fernet.decrypt(encryptedstr)

#purely for the looks
print("Decrypting string [#      ]")
time.sleep(0.1)
print("Decrypting string [##     ]")

print("Decrypting string [####   ]")
time.sleep(0.1)
print("Decrypting string [#####  ]")
time.sleep(0.1)
print("Decrypting string [#######]")

print("Done")
